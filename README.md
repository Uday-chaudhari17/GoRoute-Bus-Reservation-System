## 🚌 GoRoute Bus Reservation System

A Flask and MySQL based online bus reservation system that allows users to search buses, reserve seats, generate tickets, manage bookings, and simulate payments through an intuitive web interface.

## ✨ Features

* 🔐 User Registration & Login
* 🚌 Bus Search and Route Selection
* 💺 Interactive Seat Booking
* 💳 Payment Simulation
* 🎫 PDF Ticket Generation
* 📜 Booking History
* 📊 Dashboard Analytics
* 📍 Live Bus Tracking
* 🕒 Timetable Management

## 🛠️ Tech Stack

* Python
* Flask
* MySQL
* HTML5
* CSS3
* JavaScript


## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/Uday-chaudhari17/GoRoute-Bus-Reservation-System.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Create a MySQL database

4. Import `bus_reservation.sql`

5. Configure `.env`

6. Run the application

```bash
python app.py
```
