# GoRoute Bus Reservation System

A Flask and MySQL based online bus reservation system that allows users to search buses, reserve seats, generate tickets, and manage bookings.

## Features

* User Registration & Login
* Bus Search
* Seat Booking
* Ticket Generation
* Booking History
* Payment Interface
* Live Bus Tracking
* Timetable Management

## Tech Stack

* Python
* Flask
* MySQL
* HTML
* CSS
* JavaScript

## Installation

1. Clone the repository
2. Install dependencies
3. Create MySQL database
4. Import `bus_reservation.sql`
5. Configure `.env`
6. Run `app.py`
